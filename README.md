# 🚚 Courier Management System (Java Desktop Application)

## 📌 Overview

This is a **Courier Management System** developed using Java. The application is designed to manage courier records efficiently, including adding, tracking, and managing courier details. It uses a MySQL database for storing and retrieving data.



## ✨ Features

* Add new courier details
* Update existing courier information
* Delete courier records
* Search and track courier status
* User-friendly graphical interface (GUI)
* Database connectivity with MySQL



## 🛠️ Technologies Used

* Java (Swing / JFrame)
* MySQL Database
* JDBC (Java Database Connectivity)
* Apache NetBeans IDE


## 🖥️ How to Run the Project

1. Install Java (JDK 8 or above)
2. Download or clone this repository
3. Import the database:

   * Open MySQL Workbench
   * Import the provided `.sql` file
4. Open the project in NetBeans (optional) OR
5. Run the application using the `.jar` file:

   



## 🗄️ Database Setup

* The database file is included as `.sql`
* Import it into MySQL before running the application
* Update database credentials in the code if needed



## 📂 Project Structure

* `src/` → Source code
* `dist/` → Compiled `.jar` file
* `database/` → SQL file
